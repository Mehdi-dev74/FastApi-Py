# AI Browser Agent

AI-powered browser automation agent with memory support and Docker integration.

## Features

- Browser automation
- Agent memory management
- Docker support
- Python-based architecture

## Project Structure

```text
.
├── agent.py
├── browser.py
├── memory.py
├── demo.py
├── Dockerfile
└── README.md
```

## Installation

Clone the repository:

```bash
git clone https://github.com/USERNAME/REPO.git
cd REPO
```

Install dependencies:

```bash
pip install -r requirements.txt
```

## Usage

Run the demo:

```bash
python demo.py
```

## Docker

Build image:

```bash
docker build -t ai-agent .
```

Run container:

```bash
docker run ai-agent
```
